# ToDoTask

The app was made with Spring Boot (Java), HTML, CSS (Bootstrap) using thymeleaf to connect the backend to the frontend.

#Tools

- Java (11)
- Spring (2.5.6)
- MySql (8.0.25)
- Bootstrap (4.6)

#App

The app makes a simple managment of tasks, creating or editing them.
